# Portfólio Git — Desenvolvimento de Software

Este repositório documenta minha evolução no uso do Git ao longo das semanas 2 e 3 da disciplina de Desenvolvimento de Software.

## Objetivo

Demonstrar domínio prático de:
- Commits semânticos
- Uso de branches
- Pull Requests
- Resolução de conflitos
- Organização de repositório

## Estrutura

- `exercicios/`
  - `ex01-commits/`: prática de commits semânticos
  - `ex02-branches/`: uso de branches e merges
  - `ex03-conflito/`: resolução de conflitos
- `projeto/hello-world/`: mini projeto usado nos testes
- `REFLEXAO.md`: análise do processo de aprendizado
