# Portfólio Git — Desenvolvimento de Software

Este repositório documenta minha evolução no uso do Git na disciplina de Desenvolvimento de Software do CIn UFPE.

## Objetivo

Demonstrar domínio prático de:

* Commits semânticos
* Uso de branches
* Pull Requests
* Resolução de conflitos
* Organização de repositório

## Estrutura

* `exercicios/`

  * `ex01-commits/`: prática de commits semânticos
  * `ex02-branches/`: uso de branches e merges
  * `ex03-conflito/`: simulação e resolução de conflitos

* `projeto/hello-world/`
  Mini projeto utilizado como base para os exercícios.

* `REFLEXAO.md`
  Registro das dificuldades, aprendizados e pontos que ainda precisam de aprofundamento.

## Observações

O foco deste repositório não está no produto final, mas no processo de versionamento. Busquei manter um histórico de commits claro e organizado, de forma que seja possível acompanhar a evolução das alterações realizadas.
